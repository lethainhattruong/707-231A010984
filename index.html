Phần 1: Lý thuyết (40 điểm)

1. Position Absolute vs Relative

Relative: Định vị tương đối so với vị trí gốc, vẫn giữ chỗ trong layout. Absolute: Định vị theo phần tử cha có position, bị tách khỏi document flow. Ví dụ: Dùng Relative cho khung ảnh và Absolute cho nút "Mua ngay" đè lên ảnh.

2. Domain, Hosting & DNS

Quy trình: Mua Domain -> Thuê Hosting -> Trỏ DNS. DNS giúp chuyển đổi tên miền thành IP. Công cụ hỗ trợ: GitHub Pages, Netlify. Rủi ro: Lộ dữ liệu nếu không có SSL/HTTPS.

3. Let, Const, Var

Var có hoisting và function scope. Let/Const có block scope. Lỗi thường gặp: Gán lại giá trị cho Const. Debug bằng F12 -> Console.

4. Event Delegation

Là kỹ thuật gán 1 listener cho cha để quản lý tất cả con. Giúp tiết kiệm bộ nhớ khi danh sách sản phẩm thay đổi động.
